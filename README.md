# Rouge-Like terminal Game in C++

## Install

```bash
$ clone git@github.com:mohfunk/Rouge-Like.git
$ cd Rouge-like
$ make # compile
$ ./rouge # run the game
```

## Commands

#### Navigation
```bash
    we - move west
    ea - move east
    no - move north
        ne - move North East
        nw - move North West
    so - move south
        se - move South East
        sw - move North West
```

#### Actions
```bash
    u <dir> - use item on <dir>
    a <dir> - attack entity on <dir>
```

#### Other
```bash
    r - Restart
    q - Quit
```
